NYC Open Data Overview

The NYC Open Data Portal provides free public access to over 3,000 datasets from NYC agencies. The Education category contains data from the NYC Department of Education (DOE) covering schools, students, and academic outcomes.

Dataset Catalog

Loading the Data Library

## ============================================================
## Load NYC Open Data library
## ============================================================

source("src/R/setup.R")
source("src/R/nyc_data.R")

# Display available datasets
datasets <- list_datasets()
print(datasets)

Schools Data

School Directory

## ============================================================
## School Directory Dataset
## Comprehensive list of all NYC public schools
## ============================================================

info <- get_dataset_info(NYC_DATA$school_directory)
cat("Dataset:", info$name, "\n")
cat("ID:", NYC_DATA$school_directory, "\n")
cat("Last Updated:", format(info$last_updated, "%Y-%m-%d"), "\n")
cat("Columns:", info$columns, "\n")

## Building Space Usage Dataset

Accessing the Data

Basic Fetch

# Fetch with automatic caching
schools <- fetch_dataset(
  dataset_id = NYC_DATA$school_directory,
  cache_name = "school_directory"
)

With Filtering (SoQL)

# Server-side filtering
brooklyn_schools <- fetch_dataset(
  dataset_id = NYC_DATA$school_directory,
  `$where` = "boro = 'K'"  # K = Brooklyn
)

Large Datasets

# Paginated fetch for large datasets
all_demographics <- fetch_large_dataset(
  dataset_id = NYC_DATA$demographic_snapshot,
  batch_size = 50000,
  cache_name = "demographics_full"
)
