NYC Education Analytics

Exploring Schools, Students, and Learning through Data

Welcome

This project explores NYC Open Data education datasets using R for statistical analysis and visualization. All analysis is written in Org-mode for literate programming and exported to Quarto for publishing.

Data Sources

We use the NYC Open Data Education Portal which provides:

  • School locations and building information
  • Student enrollment and demographics
  • Test scores and academic performance
  • Graduation and dropout rates
  • Program participation data

Getting Started

# Load required packages
source("src/R/setup.R")

# Fetch school directory data
schools <- fetch_dataset(NYC_DATA$school_directory)

# Quick overview
glimpse(schools)

# Welcome

This project explores NYC Open Data education datasets using R for statistical analysis and visualization. All analysis is written in Org-mode for literate programming and exported to Quarto for publishing.


## Data Sources

We use the [NYC Open Data Education Portal](https://data.cityofnewyork.us/browse?category=Education) which provides:

-   School locations and building information
-   Student enrollment and demographics
-   Test scores and academic performance
-   Graduation and dropout rates
-   Program participation data


## Getting Started

    # Load required packages
    source("src/R/setup.R")
    
    # Fetch school directory data
    schools <- fetch_dataset(NYC_DATA$school_directory)
    
    # Quick overview
    glimpse(schools)
