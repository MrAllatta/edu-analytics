* NYC Open Data Overview

The [[https://opendata.cityofnewyork.us/][NYC Open Data Portal]] provides free public access to over 3,000 datasets from NYC agencies. The Education category contains data from the NYC Department of Education (DOE) covering schools, students, and academic outcomes.

* Dataset Catalog

#+NAME: load-library
#+BEGIN_SRC R

## ============================================================

* Accessing the Data

** Basic Fetch

#+BEGIN_SRC R :eval no
# Fetch with automatic caching
schools <- fetch_dataset(
  dataset_id = NYC_DATA$school_directory,
  cache_name = "school_directory"
)
#+END_SRC

** With Filtering (SoQL)

#+BEGIN_SRC R :eval no
# Server-side filtering
brooklyn_schools <- fetch_dataset(
  dataset_id = NYC_DATA$school_directory,
  `$where` = "boro = 'K'"  # K = Brooklyn
)
#+END_SRC

** Large Datasets

#+BEGIN_SRC R :eval no

all_demographics <- fetch_large_dataset(
